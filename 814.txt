<<@Component >>
어노테이션을 이용하면 Bean Configuration 파일에 Bean을 따로 등록하지 않아도 사용할 수 있다.
빈 등록자체를 빈 클래스 자체에다가 할 수 있다는 의미이다.
기본적으로 타입기반의 자동주입 어노테이션이다. 
@Autowired, @Resource와 비슷한 기능을 수행한다고 할 수 있겠다.
xml 설정파일에서 설정하는 방법이다.
-----------------------------------------------------------------------------
@Component
public class DESKTop {}
--> DeskTop으로 써야 애러 안남

System.out.println("DESKTop:"+context.getBean("DESKTop"));
